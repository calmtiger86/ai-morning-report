위키피디아, AI로 작성한 문서 전면 금지 — "LLM으로 글 쓰는 것 자체가 위반"

세계 최대 백과사전 위키피디아가 AI 생성 콘텐츠를 공식 금지했습니다. 영어 위키피디아 편집자들이 44대 2로 "LLM을 사용해 문서를 작성하거나 다시 쓰는 것은 금지"라는 새 규정을 통과시켰습니다. 이전에는 "처음부터 AI로 문서를 만들지 말 것"이라는 권고 수준이었지만, 이제는 부분 수정도 금지됩니다. AI가 만든 부정확한 정보가 위키피디아에 올라가고, 이를 다시 AI 학습 데이터로 쓰는 '악순환'을 막기 위한 조치입니다.
더보기접기
#위키피디아#AI규제#콘텐츠정책
이번 금지 조치의 배경에는 'TomWikiAssist'라는 AI 에이전트 사건이 있습니다. 이 자율 AI가 24시간 쉬지 않고 여러 문서를 작성·편집하며 인간 편집자들이 감당할 수 없는 규모의 혼란을 일으켰습니다.

예외 사항: 두 가지 경우에만 AI 사용이 허용됩니다. (1) 자신이 쓴 글의 맞춤법·문법 교정 제안, (2) 번역 초안 작성. 단, 두 경우 모두 반드시 사람이 정확성을 검토해야 합니다.

나에게 미치는 영향: 위키피디아를 참고 자료로 쓸 때 AI 오염 콘텐츠를 걱정할 필요가 줄어듭니다. 반면, 블로그나 콘텐츠 플랫폼도 비슷한 정책을 도입할 가능성이 높아지므로 AI로 글을 쓸 때 더욱 신중해야 합니다.

구글 Gemini, ChatGPT 대화 기록 통째로 가져오기 기능 출시

구글이 Gemini 사용자들이 ChatGPT나 Claude에서 쌓은 대화 기록을 그대로 가져올 수 있는 기능을 공개했습니다. "다른 AI 쓰다가 Gemini로 갈아타고 싶은데 처음부터 다시 설명하기 귀찮다"는 분들을 위한 기능입니다. ChatGPT에서 데이터를 ZIP 파일로 내보내면 Gemini에 업로드할 수 있고, '기억(Memory)' 기능으로 내 취향과 맥락을 그대로 이전할 수도 있습니다. 무료 사용자도 이용 가능하며, 하루 5개 ZIP 파일(각 5GB)까지 업로드됩니다.
더보기접기
#Gemini#ChatGPT#AI이동성
두 가지 가져오기 방식:
1. 대화 가져오기: ChatGPT/Claude에서 ZIP 파일 내보내기 → gemini.google.com/import에 업로드 → 이전 대화가 Gemini 사이드바에 표시됩니다.
2. 기억 가져오기: Gemini가 제공하는 프롬프트를 기존 AI에 붙여넣기 → 생성된 요약을 다시 Gemini에 붙여넣기 → 내 취향과 맥락이 즉시 적용됩니다.

왜 중요한가: 미국 모바일 시장에서 ChatGPT 점유율이 69%→45%로 하락하고 Gemini가 15%→25%로 상승하는 상황에서, 구글이 적극적으로 사용자 유치에 나선 것입니다. 단, 가져온 대화는 구글 모델 학습에 사용될 수 있으니 민감한 정보는 주의하세요.

지역 제한: EU, 스위스, 영국에서는 아직 사용 불가합니다.

Claude, 이제 내 맥북을 직접 조작한다 — 'Computer Use' 정식 출시

Anthropic이 Claude가 사용자의 맥 컴퓨터를 직접 제어할 수 있는 'Computer Use' 기능을 출시했습니다. 마우스 클릭, 키보드 입력, 스크롤, 앱 전환까지 Claude가 대신 해줍니다. 예를 들어 "이 PPT를 PDF로 변환해서 오늘 회의 캘린더에 첨부해줘"라고 말하면, Claude가 알아서 파일을 열고, 변환하고, 캘린더 앱에 첨부합니다. 아이폰의 'Dispatch' 앱과 연동하면 외출 중에도 집 컴퓨터에 작업을 맡길 수 있습니다.
더보기접기
#Claude#에이전트#자동화
작동 방식: Claude가 화면을 스크린샷으로 캡처 → 이미지를 분석해 다음 행동 결정 → 클릭, 타이핑, 스크롤 실행 → 반복. 전용 앱 연동(Google Workspace, Slack)이 있으면 우선 사용하고, 없으면 직접 마우스/키보드 조작으로 전환합니다.

안전장치: 사용자가 허용한 앱만 접근 가능, 언제든 중단 가능, 금융·의료 앱 자동 제외, 주식 거래나 민감 데이터 입력 차단.

가격: Claude Pro(월 $20) 또는 Claude Max 구독자 대상, 현재 macOS만 지원하며 Windows는 추후 지원 예정.

활용 예시: 아침 브리핑 자동 발송, 코드 리뷰 자동화, 이메일 모니터링 후 자동 응답 등 반복 업무를 맡길 수 있습니다.

ChatGPT 파워유저의 비밀 — "메타 프롬프트"로 3배 좋은 답변 얻기

이런 분께 추천: ChatGPT를 쓰는데 원하는 답이 안 나와서 답답한 분. Reddit의 400만 ChatGPT 커뮤니티에서 가장 많은 추천을 받은 팁은 바로 '메타 프롬프트'입니다. 핵심은 "답변하기 전에 먼저 나한테 질문해"라고 시키는 것. 이렇게 하면 AI가 추측하지 않고 정확한 맥락을 파악해서 훨씬 나은 결과물을 줍니다. 복잡한 프롬프트 엔지니어링 없이도 바로 적용 가능한 실전 팁입니다.
더보기접기
바로적용 무료 #프롬프트 #생산성 #ChatGPT
메타 프롬프트 공식:
"시작하기 전에, 더 좋은 답변을 위해 필요한 질문들을 먼저 해줘. 최대한 자세하게."

왜 효과적인가: 이 프롬프트 없이는 AI가 "합리적으로 들리는" 내용을 지어내서 자신 있게 말합니다. 질문을 먼저 하게 하면 가정을 줄이고 실제 필요에 맞는 답을 얻습니다.
